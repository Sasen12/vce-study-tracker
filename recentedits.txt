Most recent edit: 2026-05-05 - Optional subject chat rooms

Summary:
- Added optional subject-specific chat rooms inside Community > Chat while keeping the existing all-school chat.
- Added an All school / Subject rooms switch so the main Community tab does not get cluttered.
- Subject rooms are generated from the subjects a student has added to their profile.
- Students can join and leave subject rooms locally; joined rooms are remembered on the device.
- Added separate room message loading and sending through new community API endpoints.
- Subject-room messages are stored in the existing community chat table with an internal room marker, so no database migration is required.
- Main chat now filters out subject-room messages so the current community chat stays clean.
- Admin delete still removes messages from both all-school and subject-room views.
- Added a one-time heads-up pop-up explaining subject rooms when Community opens after the update.
- Daily chat minutes apply across both all-school chat and subject rooms.
