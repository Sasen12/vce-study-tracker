Previous edit: 2026-05-05 - VCE Weakness Coach upgrade

Summary:
- Added a shared frontend helper layer at utils/vceCoach.ts for VCE-specific study intelligence.
- Added Home dashboard return-loop features:
  - Global search across notes, coach answers, saved questions, calendar events, files, mistakes, and flashcards.
  - VCE Weakness Coach card that detects the weakest subject/topic using weekly goals, study sessions, saved questions, events, and mistake-log evidence.
  - SAC Panic Mode dialog that builds and saves a survival plan from subject, topic, date, notes, saved questions, sessions, and previous mistakes.

  - Added a Tools mode beside Forge, Battle, and Saved.
  - Added Exam Mode for timed mini-tests using the existing AI question generator and examiner-style answer feedback.
  - Added Command Term Trainer for Define, Describe, Explain, Analyse, Evaluate, Justify, and Compare.
  - Added Mistake Log view with manual mistake capture, auto-saving weak checked answers, delete, and "make card" actions.
  - Added Flashcard Forge to turn notes, coach answers, class notes, and mistake logs into review cards.
  - Added flashcard review with reveal/hide and next/previous controls.
  - Added a "Mistake" save action directly on checked Forge answers.
- Added a simple Study focus filter:
  - Timer mode can hide extra study tools during a running session.
  - Students can toggle the filter on/off from the timer.
